# Using-Bootstrap

Clone Of News Week Using Bootstrap 4

## Tools Used

#### Pug Templating Engine

- https://pugjs.org

## Getting Started

**Dev Mode**:

- `npm run start` to start dev server

**Build Production**:

- `npm run build` to build to `dist/` directory
